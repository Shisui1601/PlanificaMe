⚡ INICIO RÁPIDO (5 MINUTOS):

1. Ir a carpeta backend:
   cd backend

2. Windows:
   install.bat

   Mac/Linux:
   bash install.sh

3. Editar .env con tus datos

4. Ejecutar:
   uvicorn main:app --reload

5. Acceder a:
   http://localhost:8000/docs

🆘 PROBLEMAS COMUNES:

   "Error: ImportError: No module named 'fastapi'"
   → Ejecutar: pip install -r requirements.txt

   "Database connection refused"
   → Verificar DATABASE_URL en .env

   "CORS error en frontend"
   → Actualizar FRONTEND_URL en Railway

   "No llegan los correos"
   → Verificar SMTP_PASSWORD (16 caracteres)
   → Activar 2FA en Google

Más ayuda: Ver archivos .md correspondientes
